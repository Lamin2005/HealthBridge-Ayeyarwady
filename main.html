စိတ်ကျန်းမာရေး

Main Character Syndrome ရှိနေလားဆိုတာကို ဘယ်လိုများ သိနိုင်မလဲ!

Main Character Syndrome

Main Character Syndrome ဆိုတာက ကိုယ်တိုင်တောင် မသိလိုက်ဘဲ ကိုယ့်မှာ ရှိနေနိုင်တဲ့ စိတ်ကျန်းမာရေး အခြေအနေ တစ်ခုပါ။ လူတော်တော်များများမှာ အခြေအနေချင်း မတူဘဲ ရှိနေတတ်ပါတယ်။ ဒီတော့ ကိုယ်တိုင်မှာရော Main Character Syndrome ရှိနေလားဆိုတာကို ဘယ်လိုများ သိနိုင်မလဲ! ဆိုတာနဲ့ ဘယ်လို လက္ခဏာတွေ ခံစားရနိုင်မလဲဆိုတာလေးကို ပြောပြပေးချင်ပါတယ်။

လူတိုင်းမှာ ကိုယ်တိုင်တောင် မသိလိုက်ဘဲ Main Character Syndrome ရှိနေနိုင်ပါတယ်။ ဒါက စိတ်ကျန်းမာရေး အခြေအနေ တစ်ခုပေါ့။ Main Character Syndrome ဆိုတာက လူတစ်ဦးဦးဟာ သူတို့က လူတိုင်းကို ဦးဆောင်နေရတဲ့သူ ဒါမှမဟုတ် သူ့ဘဝမှာ သူက အရေးအပါဆုံးသောသူ၊ အရာရာကို စွမ်းဆောင်နိုင်သောသူရယ်လို့ ထင်မှတ်နေတတ်ကြသူ တွေပါ။

ဒီစကားရပ်လေးက ရုပ်ရှင်တွေ ပုံပြင်တွေထဲကနေတဆင့် မွေးဖွားလာတဲ့ စကားလုံးလေးပါ။ ဇာတ်လမ်း တစ်ခုမှာ အဓိက ဇာတ်ဆောင်နေရာက ပါဝင်ရသူလို့ ကိုယ့်ကိုယ်ကိုယ် ထင်မှတ်ခြင်းလို့ပဲ အဓိပ္ပာယ်ဖွင့်ဆိုရပါတော့မယ်။ ရုပ်ရှင်တစ်ခုခုကို ကြည့်တဲ့အခါ အဓိကဇာတ်ဆောင် (မင်းသား၊ မင်းသမီး) နဲ့ ဇာတ်ပို့ (အခြားသော အကူသရုပ်ဆောင်တွေ) ပါတတ်ကြတယ်လေ။ ဒီလိုမျိုးထဲမှာ အဓိက ကျသူက မင်းသား မင်းသမီးဟုတ်။

ဒီလိုပါပဲ။ Main Character Syndrome ရှိသူတွေကလည်း သူတို့က ဘဝမှာ ဇာတ်ပို့ဇာတ်ရံ အနေနဲ့ နေထိုင်ခြင်း မဟုတ်ဘဲ အဓိက ဇာတ်ဆောင်အနေနဲ့ ရှင်သန်နေထိုင်ရသူ အခြားသူတွေထက် သူက အရေးပါတဲ့သူလို့ ယူဆထားကြသူတွေပါ။ ဒါကြောင့်ပဲ အရာရာတိုင်းက သူ့ကို ဦးတည်ပြီး ဖြစ်နေတယ်။ သူပါမှ ပြီးမယ်ဆိုတဲ့ အထင်ကို အမြဲ လက်ကိုင်ထား လုပ်ဆောင်နေကြတာပေါ့။

လူတွေရဲ့ အရေးပါချင်တဲ့ စိတ်ကနေ အရင်းခံပြီး ဖြစ်လာတဲ့ စိတ်ပိုင်းဆိုင်ရာ အခြေအနေ တစ်ခုပေါ့။ ဒါဆို ကိုယ်တိုင်မှာရော Main Character Syndrome ရှိနေလားဆိုတာကို ဘယ်လိုများ သိနိုင်မလဲ! ဆိုတာကို သိချင်နေပြီလား…. ဒါဆိုရင် အတူတူ ဆက်ကြည့်လိုက်ရအောင်နော်။

Main Character Syndrome ရှိနေလားဆိုတာကို သိနိုင်မယ့် အချက်လေးတွေကတော့ တော်တော်များများ ရှိပါတယ်။ ဒါပေမယ့် အတွေ့ရများတဲ့ အချက်လေးတွေကတော့ အခုလိုအချက်လေးတွေပါ။

သူ့ကိုယ်သူ အဓိကထားတယ်
အရာရာတိုင်းမှာ သူ့ကိုယ်သူ ဦးစားပေးပြီး တွေးတတ်ကြပါတယ်။ ကိုယ့်ကိုယ်ကို ဂရုစိုက်တာ ကောင်းပေမယ့် သူကကျတော့ ကိုယ့်တစ်ဖို့တည်းကြည့်တဲ့ဘက်ကို ပိုသွားပါတယ်။

လူတိုင်းက သူ့ကိုပဲ အရေးစိုက်နေတာ လိုချင်တယ်
ပတ်ဝန်းကျင်ကသူတွေက သူ့ကိုပဲ အာရုံစိုက်နေတာကို လိုချင်တယ်။ လူတိုင်းရဲ့ ကိစ္စတိုင်း ဖြစ်တည်မှုတိုင်းက သူ့အပေါ်မှာပဲ အဓိက မူတည်နေစေချင်သူမျိုးပါ။ လူတွေ အာရုံစိုက် မခံရရင်လည်း attention ရအောင် ယူတတ်ကြတဲ့ attention seeker လေးတွေပါ။

Drama တအားချိုးတတ်တယ်
Main Character Syndrome ရှိနေသူတွေက Drama တအားချိုးတတ်ကြသူတွေပါ။ တစ်ယောက်ယောက်ကို ထိခိုက်အောင် သူကိုယ်တိုင်က လုပ်လိုက်ပေမယ့် သူက ခံလိုက်ရသူလိုမျိုး ပြောတတ် ငိုတတ်ကြသူတွေပေါ့။

လက်တွေ့ဘဝနဲ့ ကင်းကွာနေတတ်တယ်
Main Character Syndrome ရှိသူမှာ တွေ့ရနိုင်တဲ့ နောက်ထပ် လက္ခဏာ တစ်ခုကတော့ လက်တွေ့ကျကျ ရှင်သန်နိုင်မှု နည်းတာပါ။ နည်းနည်းလောက်ပဲ အောင်မြင်ရင်တောင် ကမ္ဘာကျော် အောင်မြင်မှုကြီးကို စွမ်းဆောင်ထားသလို ပြုမူတတ်ကြသူတွေပါ။ လက်တွေ့ဘဝကို ကင်းကွာနေရင်း ကြာလာတော့ လက်တွေ့ဘဝကို မေ့သွားတော့တာပေါ့။

ကိုယ့်ကိုယ်ကိုယ် ဘယ်သူဆိုတာ မေ့ထားတတ်တယ်
သူတို့က သူတို့ကိုယ်သူတို့ ရုပ်ရှင်တွေ ကာတွန်းကားလေးတွေထဲက စိတ်ထားကောင်းတဲ့ မင်းသမီးလေးတွေလို ထင်မှတ်ထားကြသူတွေပါ။ သူရဲ့ လက်တွေ့ဘဝက သူဆိုတာကို မေ့ထားရင်း လူတိုင်း သတ္တဝါတိုင်းက သူတို့ကို ချစ်ကြတယ်။ သူက စင်ဒရဲလားလေးလိုပဲ ချစ်စရာကောင်းသူလို့ ထင်မှတ်ထားကြသူမျိုးပါ။

ပြဿနာကို လက်တွေ့ကျကျ မဖြေရှင်းဘဲ ခပ်ရိုရိုတွေးတတ်တယ်
ပြဿနာတစ်ခုခု ဖြစ်ပြီဆိုရင် လက်တွေ့ကျကျ ဖြေရှင်းနိုင်မယ့် နည်းလမ်းအစား ရုပ်ရှင်တွေထဲကလို ကားဘီးပေါက်ရင် မင်းသားက ပေါ်လာပြီး ကားဘီးလဲပေးမယ်ဆိုတာမျိုး တွေးတတ်သူတွေပါ။ တကယ်ကတော့ ကားဘီးပေါက်ရင် ကားဘီးအပိုကိုလဲ... တကယ်ကတော့ ကားဘီးပေါက်ရင် ကားဘီးအပိုကို လဲရမှာ၊ အပိုမရှိရင် သွားဝယ်ရမှာ၊ ဒါမှမဟုတ် အခွင့်အရေး မရှိရင် မဖြေရှင်းနိုင်တာမျိုးမျိုး ဖြစ်နိုင်ပါတယ်။ ဒါပေမယ့် Main Character Syndrome ရှိသူကတော့ အမြဲတမ်း စိတ်ကူးယဉ်မှုထဲမှာပဲ နေနိုင်သူ ဖြစ်တာပါ။

အခြားသူတွေရဲ့ ခံစားချက်ကို သတိမထားတတ်ဘူး
Main Character Syndrome ရှိသူတွေဟာ သူတို့ကိုယ်သူတို့ကို အဓိကထားပြီး လူနာမကျင်စရာတွေကို လုပ်တတ်ကြပါတယ်။ သူတို့တုန်းကလည်း ဒီလိုလုပ်ခဲ့တယ်ဆိုပြီး အားထားနိုင်သူတွေက သူ့လို မလုပ်ဘူးဆိုတာကို နားမလည်နိုင်တာမျိုးပါ။ သူ့ဘဝကို သူက အဓိကထင်လွန်းတဲ့အတွက် လူတွေရဲ့ ခံစားချက်ကို သိပ်ပြီး နားမလည်နိုင်ကြဘူး။

ဘယ်လိုရော ကိုယ်မှာ Main Character Syndrome ရှိလား?
အထက်မှာပြောခဲ့တဲ့ လက္ခဏာတွေကို တဖြည်းဖြည်း စစ်ကြည့်ရင်း ကိုယ့်မှာ ရှိလား မရှိဘူးလား ဆိုတာကို သိနိုင်မှာပါ။ ဒီ Syndrome ဟာ တစ်ခါတစ်ရံမှာ ကိုယ့်မှာရှိနေပေမယ့် သိပ်လည်း ပြင်းထန်မှု မရှိနိုင်ပါဘူး။

တကယ်လို့ ကိုယ့်မှာ ဒီ Syndrome ရှိနေလို့ ကိုယ့်ဘဝကို ကောင်းမွန်အောင်ပြုလုပ်ချင်တယ်ဆိုရင်တော့ တဖြည်းဖြည်းဖြေရှင်းကြည့်ဖို့လိုပါလိမ့်မယ်။

Main Character Syndrome ကို ဘယ်လိုဖြေရှင်းကြမလဲ?

လူတွေရဲ့ ခံစားမှုကို နားလည်ရန်
ကိုယ့်ရဲ့ အပြုအမူတွေဟာ အခြားသူတွေကို ထိခိုက်စေမှာလား၊ တကယ်ရော သူတို့အတွက် အဆင်ပြေလား၊ ကိုယ့်လုပ်ဆောင်မှုတွေက သူတို့ဘက်ကနေ ကြည့်ရင် မှန်မမှန်ဘူးလား ဆိုတာကို စဉ်းစားကြည့်ဖို့လိုပါတယ်။

လက်တွေ့ကျကျ ဖြေရှင်းတတ်ဖို့
ကိုယ့်ဘဝထဲမှာ ဖြစ်လာနိုင်မယ့် ပြဿနာတွေကို ရုပ်ရှင်လိုမျိုး ထွက်လွှတ်မနေပါနဲ့။ တကယ်လက်တွေ့ကျကျ ဘယ်လိုဖြေရှင်းရမယ်ဆိုတာကို တွေးကြည့်ပြီး ဖြေရှင်းနိုင်ဖို့ အရင်မှန်းဆကြည့်ပါ။

ကိုယ့်ကိုယ်ကိုယ် အဓိကမထားလို့
လူတိုင်းက ဇာတ်ဆောင် ဖြစ်ချင်တယ်။ ဒါပေမယ့် ဘဝမှာ လူတိုင်းက ဇာတ်ပို့ပါပဲ။ ကိုယ့်ဘဝမှာ ကိုယ်က အဓိက ဇာတ်ဆောင်ပေမယ့် အခြားသူတွေရဲ့ ဘဝထဲမှာ ကိုယ်က ဇာတ်ပို့ဖြစ်နေရပါတယ်။ ဒါကို နားလည်နိုင်ဖို့ လိုပါတယ်။

တစ်ခြားသူတွေကိုလည်း အရေးထားဖို့
ကိုယ့်ဘဝအတွက် ကိုယ်တော်တော်လေး အရေးထားတတ်သလို တစ်ခြားသူတွေလည်း သူတို့ဘဝမှာ အရေးကြီးသူတွေပါဆိုတာကို နားလည်နိုင်ဖို့ လိုပါတယ်။ သူတို့ရဲ့ ခံစားမှုတွေကို တန်ဖိုးထားနိုင်ဖို့ လိုပါတယ်။

Main Character Syndrome ရှိနေရင် တခြားသူတွေကို မထိခိုက်စေဖို့
ကိုယ့်လိုက်လုပ်နေတဲ့ အပြုအမူတွေက တခြားသူတွေကို မထိခိုက်စေဖို့လိုပါတယ်။ ကိုယ့်အတွေးအမြင်တွေဟာ တခြားသူတွေကို ဦးစားပေးနိုင်ဖို့ လိုပါတယ်။

Main Character Syndrome ရှိတယ်ဆိုတာက ကိုယ့်မှာ အနည်းငယ်ရှိနေရင်တော့ မကောင်းတာ မဟုတ်ပါဘူး။ တချို့က သူတို့ဘဝမှာ ရည်မှန်းချက် တစ်ခုခု တည်ဆောက်နိုင်ဖို့ Main Character Syndrome ကိုအသုံးချနိုင်ပါတယ်။ ဒါပေမယ့် တခြားသူတွေကို ထိခိုက်စေမှာ ဖြစ်လာမယ်ဆိုရင်တော့ ပြုပြင်သင့်ပါတယ်။

ကိုယ့်ဘဝမှာ ကိုယ့်ကိုယ်ကိုယ် တန်ဖိုးထားတတ်ဖို့လိုပေမယ့် တခြားသူတွေကိုလည်း တန်ဖိုးထားဖို့လိုပါတယ်။ ကိုယ့်ရဲ့ အတွေးအမြင်တွေအားလုံးဟာ မှန်တယ်လို့ မထင်သင့်ပါဘူး။ တခြားသူတွေရဲ့ အမြင်တွေကိုလည်း နားထောင်တတ်ဖို့လိုပါတယ်။

ဘဝမှာ တကယ်တမ်း Main Character ဖြစ်နေသူလည်း ရှိနိုင်ပါတယ်။ ဒါပေမယ့် တကယ် အဓိက ဇာတ်ဆောင်ဖြစ်ဖို့ဆိုရင် တခြားသူတွေကို အရေးထားနိုင်တဲ့ ဇာတ်ဆောင် ဖြစ်ဖို့လိုပါတယ်။

    အခြားရောဂါများ

    မျက်စိကျန်းမာရေး ကျန်းမာစွာ စားသောက်ခြင်း ကင်ဆာရောဂါများ ဆီးချို သွေးချို အိပ်စက်ခြင်း

    အမေးများသောမေးခွန်းများ:

    "စိတ်ကျန်းမာရေးဆိုတာ ကျွန်ုပ်တို့ရဲ့ သိမှု၊ ခံစားမှု၊ စိတ်ပိုင်းဆိုင်ရာ၊ လူမှုရေးဆိုင်ရာ ကျန်းမာရေးကို ဆိုလိုတာပါ။ ဒါက ကျွန်ုပ်တို့ရဲ့ အတွေး၊ ခံစားချက်၊ လူမှုဘဝနဲ့ ဘဝဖြတ်သန်းပုံတွေကို သက်ရောက်မှုရှိပါတယ်။

    ကျန်းမာတဲ့ စိတ်ဆိုတာ ရောဂါမရှိရုံသက်သက် မဟုတ်ပါဘူး။ စိတ်ကျန်းမာရေး ကောင်းမွန်တဲ့အခါ စိတ်ပိုင်းဆိုင်ရာရော၊ ခံစားချက်ပိုင်းဆိုင်ရာပါ ကောင်းမွန်နေမှာပါ။ ပျော်ရွှင်နေရင် စိတ်ကျန်းမာတာပဲ မဟုတ်လားလို့ ပြောကြပေမယ့် စိတ်ကျန်းမာရေးကောင်းဖို့ တစ်ချိန်လုံး ပျော်ရွှင်နေစရာ မလိုပါဘူး။"

    ""စိတ်ကျန်းမာရေးက ကျွန်ုပ်တို့ ပတ်ဝန်းကျင်ကို ဘယ်လိုမြင်ပြီးတုံ့ပြန်သလဲဆိုတာပေါ် သက်ရောက်နေတဲ့အတွက် အရေးကြီးပါတယ်။ ကျွန်ုပ်တို့ရဲ့ အတွေးတွေ၊ ခံစားချက်တွေ၊ လူမှုဆက်ဆံရေးတွေ၊ ဘဝအပေါ် ရှုမြင်ပုံတွေကို စိတ်ကျန်းမာရေးက လွှမ်းမိုးထားပါတယ်။ ဒီနှစ်ပိုင်းတွေမှာ အလုံးစုံကျန်းမာရေးကို ပိုပြီးအာရုံထားလာကြပါတယ်။ အလုံးစုံကျန်းမာရေးမှာ စိတ်ကျန်းမာရေးလည်း ပါပါတယ်။ ဒါက ကိုယ်ကျန်းမာရေးအပေါ် သက်ရောက်မှု ရှိပါတယ်။ ဥပမာ စိတ်ကျရောဂါက လေဖြတ်နိုင်ခြေ၊ နှလုံးရောဂါနဲ့ အမျိုးအစား ၂ဆီးချိုရောဂါ ဖြစ်နိုင်ခြေကို များစေတယ်လို့ လေ့လာချက်တွေအရ သိရှိထားပါတယ်။"

    "CDCရဲ့ အဆိုအရ ၅၀ ရာခိုင်နှုန်းသော လူတွေက သူတို့ ဘဝတစ်ကွေ့ကွေ့မှာ စိတ်ကျန်းမာရေး ပြဿနာကို ခံစားဖူးပါတယ်တဲ့။ အာရှမှာဆိုရင် လူပေါင်း ၄၅၀ မီလီယမ်လောက်က စိတ်ကျန်းမာရေးပြဿနာကို ခံစားနေရပါတယ်။ စိတ်ကျန်းမာရေးပြဿနာတွေက သင်ထားတာတွေထက် ပိုပြီး အဖြစ်များပါတယ်။ နာတာရှည် စိတ်ကျန်းမာရေး ပြဿနာမရှိသည့်တိုင် စိတ်ကျန်းမာရေးအခြေအနေက အချိန်နဲ့ အမျှ ပြောင်းလဲနိုင်ပါတယ်။ ဒါကြောင့် ထူးခြားတဲ့ လက္ခဏာတွေခံစားနေရရင် စိတ်ကျန်းမာရေး အထူးကုနဲ့ တိုင်ပင်ဆွေးနွေးသင့်ပါတယ်။"

    "စိတ်ကျန်းမာရေး ပြဿနာတွေကို ဖြစ်စေတဲ့ အကြောင်းရင်း အများကြီး ရှိပါတယ်။ ဖြစ်နိုင်ခြေများစေတဲ့ အကြောင်းရင်းတွေကတော့ -ကလေးဘဝမှာ စိတ်ဒဏ်ရာ၊ အနိုင်ကျင့်တာ၊ လျစ်လျူရှုတာ ခံစားရတာ -ကင်ဆာ၊ ဆီးချိုရောဂါလိုမျိုး နာတာရှည် ရောဂါတစ်ခုခု ရှိနေတာ -မိသားစုထဲမှာ စိတ်ကျန်းမာရေး ပြဿနာခံစားရတဲ့သူရှိနေတာ -ဦးနှောက်ထဲက ဓာတုပစ္စည်းတွေ မမျှတတာ -အလုပ်ပြုတ်တာ၊ ချစ်ခင်ရတဲ့သူ သေဆုံးတာလိုမျိုး စိတ်ထိခိုက်စရာ အကြောင်းအရာတွေ ရှိနေတာ -အထီးကျန်တာ စတာတွေပဲ ဖြစ်ပါတယ်။"

    "စိတ်ကျန်းမာရေးပြဿနာတွေမှာ လက္ခဏာတွေ အမျိုးမျိုးရှိနိုင်ပါတယ်။ အောက်ပါ လက္ခဏာတွေ ခံစားနေရရင်တော့ တတ်ကျွမ်းနားလည်သူနဲ့ ပြသဖို့ လိုပါတယ်။ -အမူအကျင့်တွေ ပြောင်းလဲလာတာ -စားသောက်ပုံ၊ အိပ်စက်ပုံတွေ သိသိသာသာ ပြောင်းလဲလာတာ -သူငယ်ချင်းတွေ၊ ပုံမှန်လုပ်နေကျအလုပ်တွေကနေ ရှောင်ချင်လာတာ -နေ့စဉ်ပြဿနာတွေ၊ စိတ်ဖိစီးမှုတွေကို မခံနိုင်တာ -ကိုယ့်ကိုယ်ကို အဆုံးစီရင်ဖို့ စဉ်းစားလာတာ -အစွန်းရောက် ခံစားချက်တွေ ဖြစ်တာ (ဥပမာ ဒေါသအလွန်ကြီးလာတာ၊ ရန်လိုတာ) -အရက်၊ မူးယစ်ဆေး စတာတွေ အလွဲသုံးစားလုပ်လာတာ -အကြောင်းရင်း မယ်မယ်ရရ မရှိဘဲ ဖျားနာတာ (ဥပမာ ခဏခဏ ခေါင်းကိုက်တာ၊ ခါးနာတာ)"

    "သူငယ်ချင်း ဒါမှမဟုတ် မိသားစုဝင်တစ်ယောက်ယောက်မှာ စိတ်ကျရောဂါ ဒါမှမဟုတ် ကိုယ့်ကိုယ်ကို အဆုံးစီရင်ချင်စိတ်တွေ ရှိနေပြီဆိုရင် အောက်ပါ နည်းလမ်းတွေနဲ့ ကူညီနိုင်ပါတယ်။ -သူတို့ကို ကိုယ်ချင်းစာတရားနဲ့ စကားပြောပေးပါ။ -သူတို့ရဲ့ ပြဿနာကို အရေးမကြီးသလို မလုပ်ပါနဲ့။ (ဘာမှမဖြစ်ပါဘူး။ အကောင်းမြင်စမ်းပါလို့ မပြောပါနဲ့။) -သင်က သူတို့ရဲ့ပြဿနာကို ကိုယ်စားဖြေရှင်းပေမယ်လို့ မထင်ပါနဲ့။ -သူတို့က တစ်ယောက်တည်း မဟုတ်ဘူးဆိုတာကို သိပါစေ။ (အကူအညီလိုရင်အမြဲရှိတယ်ဆိုတာကို သိပါစေ။) -တတ်ကျွမ်းနားလည်သူနဲ့ ပြသဖို့ တိုက်တွန်းပါ။"
